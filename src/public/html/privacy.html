API Privacy Policy

We collect personal data for the following purposes:

How Personal Data is Collected

Personal data is collected through the following means:

Personal data is stored on secure servers and protected with appropriate security measures. We take steps to protect personal data from unauthorized access or misuse.

User Rights

Users have the following rights in relation to their personal data:

Changes to the Privacy Policy

We may update this privacy policy from time to time. We will notify users of any changes through the API or by email. Users can contact us with any questions or concerns about this privacy policy by emailing privacy@itsfishy.xyz
